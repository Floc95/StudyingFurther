# Studying Further

## Documentation utilisateur

### Présentation

L'application **Studying Further** permet de simplifier et d'améliorer les échanges entre les professeurs et les élèves. 

Elèves et professeurs peuvent mettre en ligne du contenu à disposition de la classe. Ils peuvent également commenter ce contenu.

Les professeurs peuvent gérer leurs classes et attribuer des "+1" aux contenus ou commentaires des élèves.

### Connexion à l'application

#### Création de compte

A la première connexion, si vous n'avez pas de compte, vous pouvez cliquer sur le bouton "Inscription" afin de vous créer un compte sur l'application.

A la création de compte vous sera demandé :

- Votre prénom
- Votre nom
- Un identifiant de connexion
- Un email valide
- Un mot de passe pour votre compte
- Le type de votre compte (Professeur ou Elève)

Une fois votre compte créé, vous pourrez vous connecter à l'application.


#### Identification

A votre première connexion vous sera demandé votre identifiant de connexion ainsi que votre mot de passe. 
**Attention, votre mot de passe est sensible à la casse.** En revanche, votre identifiant ne l'est pas.

L'identification de votre compte dans l'application n'est demandé qu'à la première connexion. L'application démarrera les fois suivante en vous connectant automatiquement, de façon transparente.

Dans le cas où vous êtes déjà identifié dans l'application et que vous souhaitez vous déconnecter pour vous reconnecter avec un autre compte, il vous suffit, sur l'écran principal de cliquer sur le bouton des options (les 3 petits points en haut à droite) et de cliquer sur l'option "Déconnexion".

### Flux d'actualité

#### Présentation

Sur l'écran principal, vous voyez toutes les publications des de votre flux d'actualité. Ce flux est composé de vos publications ainsi que celles de vos classes. 

Sur cet écran, vous pouvez ainsi lire les publications, voir le nombre de commentaire mais aussi attribuer des "+1" aux contenus si vous êtes professeur.

#### Détails d'un contenu

Pour voir le détails d'un contenu (i.e. voir le contenu complet, avec les commentaires), il vous suffit de cliquer sur un contenu dans votre flux d'actualité.

Cela ourira un écran dédié au contenu qui vous affichera le contenu mais également les commentaires de ce contenu. 
Si vous êtes professeur, en dessous de chaque commentaire est affiché un bouton de "+1" qui permet d'attribuer un "+1" à un commentaire. 
En la fin de la liste des commentaires, vous avez un champ de saisie afin de pouvoir ajouter un bon commentaire au contenu.

#### Ajout de contenu

Sur l'écran principal, vous avez un bouton qui vous permet d'ajouter un contenu.

En cliquant dessus, vous arrivez sur une interface dédié à la création de contenu. Vous pouvez ainsi rédiger votre contenu, ajouter un image s'il vous le souhaitez et choisir la classe à qui est adressé ce message. 

Une fois le contenu rédigé et les différents champs renseignés, vous pouvez cliquer sur le bouton de validation qui postera votre contenu.

### Gestion des classes

#### En tant que professeur

Pour gérer vos classes, il vous suffit de cliquer sur le menu d'options sur l'écran principal (les trois petits points en haut à droite). 

Cela vous ammène sur un écran vous montrant toutes les classes que vous avez. Sur cet écran vous pouvez ajouter une classe ou en supprimer.

En cliquant sur une classe, vous pouvez voir la liste des élèves de cette classe. D'ici, vous pouvez ajouter ou supprimer un élève de la classe.

#### En tant qu'élève

En tant qu'élève l'écran de gestion des classes vous montre dans quelles classes vous êtes présent.

Cependant, aucune action de votre part ne peux être faite. Seul les professeurs peuvent agir sur les classes.

### Notifications

L'application **Studying Further** intègre une gestion des notifications. Vous allez ainsi recevoir des notifications sur votre téléphone à chaque fois qu'une personne commentera une de vos publications. Vous serez ainsi tenu à informer des différentes réponses que les utilisateurs feront sur vos contenus.

### Mode hors connexion

Une gestion de mode "Hors connexion" est présente dans l'application. 

Grâce à cette fonctionnalité, vous pouvez consulter les contenu prédemment chargés dans l'application quand vous n'avez pas d'accès internet. 

Du fait d'être déconnecté d'internet, vous ne pourrez en revanche que consulter les contenus. Vous ne pourrez ni poster de nouveau contenu ou ajouter de commentaires.
